# Apideck (apideck)
Apideck is the leading Unified API that doesn't store sensitive customer data. Build and maintain native integrations into your product with an exceptional Developer Experience. Apideck provides unified APIs for accounting, CRM, HRIS, ATS, file storage, ecommerce, POS, issue tracking, SMS, webhooks, and more, connecting 200+ downstream integrations.
